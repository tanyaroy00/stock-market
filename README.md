<h2>Description</h2>
<p>This project aims to develop a web-based Stock Market Visualizer that enables users to fetch real-time and historical stock data, visualize price trends, compute basic market indicators, and observe short-term price behavior. The system is designed to be simple, interactive, and beginner-friendly, serving as an educational tool for understanding stock market fundamentals.</p>
